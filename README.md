## Qiankun+Vite+Lerna Lite+Npm Workspace

## 安装依赖

### 全局安装 `Lerna Lite`

```bash
# 卸载 lerna
npm uninstall -g lerna

# 安装 lerna-lite
npm install -g @lerna-lite/cli
```

### 安装项目依赖

```bash
npm install
```
